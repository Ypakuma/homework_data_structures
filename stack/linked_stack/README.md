## Linked Stack
#### These files include some  fundamental functions about linked stack.
- LinkedStackInit
 - create a new linked stack and initialize it.
- LinkedStackEmpty
 - judge a stack is empty or not.
- LinkedStackPush
 - push a element into the stack.
- LinkedStackPop
 - pop a element out of the stack.
- LinkedStackDestroy
 - destroy a stack.

## 链栈
#### 这些文件包括了链栈的一些基本操作函数。

- LinkedStackInit
 - 创建一个链栈并将其初始化。
- LinkedStackEmpty
 - 判断一个链栈是否为空。
- LinkedStackPush
 - 让一个元素入栈。
- LinkedStackPop
 - 让一个元素出栈。
- LinkedStackDestroy
 - 销毁链栈。
